How it works

  1. Register kids under Children — link parents, set a family PIN (4–6 digits), default room, allergies.
  2. Check-In Station — search by last name, PIN, or phone · select kids · assign rooms · print labels.
  3. Labels show name, room, and a large pickup code parents must present at release.
  4. Secure Pickup — staff enters the code · confirms the child · checks out. Parents can get email alerts.
  5. Room Board — teachers see who’s in their room in real time.

Tip: give each family a memorable PIN (last 4 of phone works great). Kids can also have their own PIN for fast lookup.
